## Prerequisites

- Node.js installed
- Backend server running on `http://localhost:3000`

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser to the URL shown (usually `http://localhost:5173`)

## Backend Configuration

The app connects to your backend API at `http://localhost:3000/api/recipes`.

Make sure your backend is running before using the frontend.

## Usage

- **Add Recipe**: Click the "Add New Recipe" button in the header
- **View Recipe**: Click "View" on any recipe card to see full details
- **Edit Recipe**: Click "Edit" on any recipe card to modify it
- **Delete Recipe**: Click "Delete" on any recipe card (with confirmation)
- **Search**: Use the search bar to filter recipes
